Data

For this project, I used two CSV files from the Florida Department of Health's Bureau of Vital Statistics. The first is a file of heart-disease deaths per 100,000 people by county. The second is a file of individuals below the poverty level per county. I assigned this data a spatial component by joining it with a shapefile of Florida county boundaries that I found on the Florida Geospatial Open Data Portal. In addition, I used a point data shapefile of hospital locations in Florida from the University of Florida.

Methods

I performed all methods by importing my data in ArcGIS

With the county shapefile:

To clip the file to just Florida counties, I used the Select By Attribute tool remove all features without 'Florida' as their state name.

With the two CSVs:

I joined both CSV files to my clipped county boundary file. Then, I manipulated the symbology for the death rate by county field and the individuals below the poverty level by county field. This created two choropleth maps which I exported by creating layouts of each layer.

To analyze the point data:

I used the Kernel Density tool in the geoprocessing tools toolbox. This gave me a raster layer indicating the areas in which there are the highest concentrations of hospitals.

To analyze the choropleth maps:

I used the Cluster Analysis and Outlier Tool on each of the layers. This tool identified spatial clusters of features with high or low values. It also calculated local Moran's I value, a z-score, a pseudo p-value, and a code representing the cluster type for each statistically significant feature.
